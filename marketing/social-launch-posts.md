### r/webdev
**Title:** Built an AI creative platform with Next.js 15, 13 API integrations, and model-aware credit pricing

Sharing DreamForgeX — an AI creative studio I built that routes requests to 30+ models across 13 providers.

**Technical highlights:**
- Next.js 15 App Router + tRPC for type-safe APIs
- Drizzle ORM + Neon Postgres
- Cloudflare R2 for asset storage
- Queue-based polling for long-running generations (Runway, Sync Labs)
- Model-aware credit system — each model has a tier-based cost
- Rate limiting per endpoint + automatic credit refunds on failure
- Edge TTS for zero-cost text-to-speech
- i18n in 6 languages

The provider adapter pattern made it easy to add new AI providers — each one implements a simple `generate()` interface.

https://dreamforgex.ai
